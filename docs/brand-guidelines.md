# ReconPilot Brand Guidelines

## Brand Summary

- Product name: ReconPilot
- Tagline: Settlement operations cockpit
- Positioning: Operations-grade reconciliation and root-cause analysis workbench for multi-channel payments

## Logo

- Primary logo: `src/main/resources/static/logo.svg`
- Favicon: `src/main/resources/static/favicon.svg`
- Usage:
  - Keep clear space around the mark equal to the mark’s corner radius.
  - Use the full-color mark on dark surfaces (sidebar) and light surfaces (cards) as provided.
  - Avoid stretching; preserve aspect ratio.

## Color Palette

Core

- Navy: `#0B1220` (primary background/ink)
- Surface: `#FFFFFF`
- Surface Alt: `#F2F5FB`

Brand accents

- Blue: `#4F8BFF`
- Blue (deep): `#2F6DF5`
- Mint: `#22C6A3`

Status

- Critical: `#E21B3C`
- High: `#F97316`
- Medium: `#8B5CF6`
- Low: `#16A34A`

## Typography

- Font family: system UI sans-serif stack (fast, native, and license-safe)
- Headings:
  - H1: 20px / 800
  - H2: 15px / 820
- Body:
  - Default: 14px / 400–650
  - Secondary: 12–13px (muted)

## Interaction Principles

- Motion: short, subtle, and functional; respect `prefers-reduced-motion`
- Focus: visible focus rings using the brand blue, keyboard-first friendly
- Density: compact but not cramped; consistent 12–16px spacing in cards and tables
